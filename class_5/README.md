## Описание каталогов и файлов в репозитории
Vagrantfile - Vagrant файл с конфигурацией автоматического поднятия двух VM клиент/сервер.
nfss.sh - файл, который содержит bash скрипт, необходимый Vagrant для выполнения на стороне сервера.
nfsc.sh - файл, который содержит bash скрипт, необходимый Vagrant для выполнения на стороне клиента.
