# Вход в Систему Без Пароля

В этом документе представлены методы для входа в систему без использования пароля. Эти методы могут быть использованы для восстановления доступа к системе в случае забытого пароля или для административного доступа.

## Методы Входа

### Использование Режима Восстановления

1. Перезагрузите компьютер и дождитесь появления меню GRUB.
2. Выберите пункт меню для входа в режим восстановления (обычно имеет название "Advanced options" или "Recovery mode").
3. В меню восстановления выберите "root" для доступа к корневой оболочке.

### Изменение Параметров Загрузки в GRUB

1. При появлении меню GRUB нажмите `e`, чтобы редактировать параметры загрузки.
2. Найдите строку, начинающуюся с `linux`, и добавьте `init=/bin/bash` в конец этой строки.
3. Нажмите `Ctrl+X` или `F10` для запуска с измененными параметрами.

### Использование Live CD/USB

1. Создайте Live CD или USB с дистрибутивом Linux.
2. Загрузитесь с созданного Live CD/USB.
3. Используйте утилиту `chroot` для доступа к файловой системе установленной системы.
